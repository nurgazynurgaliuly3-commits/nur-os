# Privacy Notes

NurOS stores personal finance, ritual, fitness, reminder, and AI chat data for the signed-in user.

## Data Stored

- Profile and fitness inputs
- Finance summaries and transaction history
- Rituals, streaks, reminders, and chat history
- Auth metadata and audit events

## User Controls

The Account screen supports:

- JSON export
- JSON import
- Manual backup
- Account deletion

## AI

If `OPENAI_API_KEY` is configured, AI chat can send the current NurOS context to the configured OpenAI-compatible API. Without an API key, NurOS uses the local rule-based AI Core fallback.

## Health Disclaimer

NurOS does not diagnose medical conditions and does not prescribe medication. Fitness, supplement, and nutrition outputs are planning aids based on user-provided data.
